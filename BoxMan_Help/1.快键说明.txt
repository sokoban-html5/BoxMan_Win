=== 快键说明 ===

Ctrl + Q  ：退出

+、-      ：增速、减速

E ：奇偶格效果开关
I ：瞬移开关
B ：正逆推模式切换

Ctrl + G：固定的目标位 -- 正逆推均使用正推的目标位
Ctrl + J：即景目标位   -- 正推以逆推现场计算目标位，逆推亦然

Z        ：撤销
X        ：重做
A/滚轮   ：撤销一步
S/滚轮   ：重做一步
退格键   ：撤销到首/暂停
空格键   ：重做到尾/暂停

ESC：重开始

*：  第 0 转
/：  旋转关卡

Page Up：  上一关
Page Domw：下一关

F1：       此帮助
F2：       更换皮肤
F3：       关卡浏览
F4：       动作编辑

Ctrl + O：打开关卡文档
Ctrl + S：保存关卡状态；但是，当关卡 XSB 尚未保存时（从剪切板导入的关卡 XSB），该功能为保存关卡 XSB 到文档 -- 所有关卡

Ctrl + V：从剪切板加载 XSB -- 允许多个关卡
Ctrl + C：当前关卡的 XSB 送入剪切板 -- 仅一个关卡

Ctrl + L：从剪切板加载 Lurd -- 区分正逆推
Ctrl + M：Lurd 送入剪切板 -- 包括正逆推

双击“状态栏”最右边一栏   ：动作的快速定位
 
旋转和速度提示上鼠标左右键 ：增减变化
